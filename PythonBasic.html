Running Python#

Running Python in VS Code#

  1. To open the ece487_wksp folder in VS Code, right-click on it and choose Open with Code from the menu.

  2. To open the Terminal window, go to View and click on Terminal.

  3. Type in python and press Enter

  4. This will launch the Python Shell, where you can input code and see the output. The Shell acts as a sandbox, allowing you to try out code snippets and get immediate feedback. For example, in the Shell, type 2+3 and press Enter, and it will display 5 on the next line.

  5. Try the following code in the Shell:

    2+3
"hello"+" world"
a = 2
print(a)

  6. The Python Shell behaves like a calculator, allowing you to perform various mathematical operations.

  7. Now, try the follownig code:

    2+2    # add two integers
2+2.0  # add an integer to a floating point number
2*3    # 2 times 3
2**2   # 2 squared
2**3   # 2 cubed
2**128 # Unlike C, integers in Python does not have a size limitation.    
2/3    # It returns 1 in C, but 0.6666666666666 in Pyton 
2/3.0  # Same as 2/3 in Python.
2//3   # modulo (remainder), same as 2%3 in C
4//3   # modulo (remainder), same as 4%3 in C

  8. To exit the Python Shell, type exit() and press Enter, or use Ctrl+z and press Enter.

Running Python in Command Prompt#

  1. Open Command Prompt or PowerShell (or Terminal on Mac and Ubuntu), and run python on Windows or python3 on Mac.

  2. This will open the Python Shell.

Working with Variables#

  1. In Python, variables store values like strings or numbers. Strings are blocks of text, such as "Stan Baek is the legendary pirate captain." Numbers can be integers or decimals.

  2. Try the the following code in Python Console.

    # a string variable
name = "Stan Baek"

  3. You can check the type in use by issuing the type() command

    type(name)

  4. It will return <class 'str'>

  5. Add another string variable:

    title = "the legendary pirate captain"

  6. You can concatenate strings using + operator:

    character = name + ": " + title
print(character)  # It will print "Stan Baek: the legendary pirate captain"

  7. Now, try combining a string with an integer:

    # Integer variable
age = 726
# This will raise an error
print(character + age)

  8. Convert the integer to a string for proper concatenation:

    print(character + " is " + str(age) + " years old.")

  9. This will print

    Stan Baek: the legendary pirate captain is 726 years old.

  10. You can also request input from the user:

    age = input("How old are you? ")
print(type(age))

# Since age is a string variable. We need to convert it to integer.
age = int(age)

# Floating-point numbers
x = 3.141
print(type(x))
print(x)  # Prints 3.141
print(int(x))  # Prints 3

  11. When creating variables, remember that single or double quotes can be used for strings, but be consistent with your choice.

Functions and Modules#

  1. Python has many built-in functions. For example:

    name = "Stan Baek"
len(name)  # Returns the lenght of the string
type(name) # Returns the type of the variable

  2. To explore more built-in functions, visit Python‚Äôs built-in functions documentation. You can also use the help() function to get descriptions:

    help(len)

  3. You can define your own functions in Python:

    def add(a, b):
    return a+b
print(add(3, 4))  # Outputs 7

Python Modules#

  1. You can add extra functionality to Python by importing modules. Modules are collections of additional functions built by others. For example, to access advanced mathematical functions, you can import the math module:

    import math
print(math.sqrt(16))    # Outputs 4.0
print(math.sin(math.pi/2))  # Outputs 1.0
print(math.cos(0))  # Outputs 1.0

  2. In Python, it‚Äôs a convention to import all required modules at the top of the script. This ensures all dependencies are loaded before they‚Äôre used and makes the code easier to understand. You can also import specific functions from a module:

    from math import sqrt, sin, pi
print(sqrt(16))
print(sin(pi/2))

  3. We can also import individual functions from a module

    from math import sqrt, sin, pi
print(sqrt(16))
print(sin(pi/2))
cos(0)   # error because cos was not imported

  4. You can also use an alias for a module:

    import math as m
m.sqrt(16)
m.sin(m.pi/2)

Conditional Statements#

  1. The if, else, and elif statements control the flow of execution based on conditions. The basic syntax is:

    if expression:
    statements
elif expression:
    statements
else:
    statements

  2. If no action is needed for a certain condition, you can use the pass statement:

    if expression:
    pass       # Do nothing.
else:
    statements

  3. Example:

    word = input("Enter a four-letter word: ")
if len(word) == 4:
    print(word, "is a four-letter word. Well done.")
elif len(word) == 3:
    print(word, "is a three-letter word.")
else:
    print(word, "is not a four-letter word.")

Loops#

  1. Let‚Äôs start with a simple while statement.

    x = 1
while x < 5:
    print(x)
    x = x + 1

  2. We can use break to exit a loop:

    while True:
    print(x)
    x -= 1
    if x == 0:
        break

  3. The for loop is commonly used in Python:

    for i in [0, 1, 2, 3, 4]:
    print(i)

  4. We can also use range():

    for i in range(5):  # Loops through 0 to 4
    print(i)
